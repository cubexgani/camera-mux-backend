# Camera Multiplexer Backend
Backend for the camera mux part. Part of a bigger project.

## Getting started:
- `git clone` this repository
- `python -m venv venv` for creating a virtual env
- `venv\Scripts\activate` or `venv/Scripts/activate` to activate venv
- `pip install` to install dependencies listed in requirements.txt
- `fastapi run .\mux-backend.py` to run the backend in production mode.
